A console-based Java program that demonstrates the use of Object-Oriented Programming concepts such as classes, objects, and methods.
In this game, a Guesser selects a number, and multiple players attempt to guess it. The Umpire compares all guesses and announces the winner(s).

Features:
-Implements core Java concepts like OOP and conditional logic.
-Interactive console-based gameplay.
-Clear role-based structure: Guesser, Players, and Umpire.
-Beginner-friendly project for understanding class interaction in Java.

How to Run:
1. Clone the repository or download the project files.
2. Open the project in any Java IDE (e.g., IntelliJ IDEA, VS Code, Eclipse).
3. Compile and run GuesserGame.java.
4. Follow the on-screen instructions to play the game.
